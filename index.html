HTML/CSS/JS 基礎

このページは div と span、リンク、画像、テーブルを例示します。

概念

  • HTML: 文書の構造（見出し・段落・画像・リンク等）を定義。
  • CSS: 見た目（色、レイアウト、余白、フォント等）を定義。
  • JavaScript: 振る舞い（インタラクション、動的更新、通信等）を定義。

DIV と SPAN の違い

DIV はブロック要素。幅いっぱいに配置され、上下に積み重なります。

一方で SPAN はインライン要素 です。文の途中の一部にスタイルを当てます。

タグ属性 / id と class

属性はタグに追加情報を与えます。例：img src="..." alt="..."。

id はページ内で一意。class は複数要素に共通スタイルを付与。

これは id="unique" の段落。

これは class="note" の段落（複数に使える）。
